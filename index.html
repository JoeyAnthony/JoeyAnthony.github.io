About

I am a software engineer from the Netherlands. I have a passion for 3D applications and autosterioscopic displays, which I have been working with for some time now. Working with new technology is what drives me and combining that with 3D graphics makes it even better! Beneath you will find some of the projects I have been working on.

Projects

XR Game Bridge

This project tries to run OpenXR applications on LeiaSR 3D displays. The idea for it is to be easy to use while also being performant.

Go to Github

Game Bridge Reshade addon

This is a Reshade addon that enables the Simulated Reality SDK in 3D applications. This combined with a 3D shader like SuperDepth3D or a mod like Geo-11 can give a cool 3D gaming experience.

Go to Github

Inverse Kinematics Implementation In Unreal Engine

Internship project at a company called RE-liON located in Enschede. I implemented an Inverse Kinematics algorithm that supports single chains and ball-socket joint constraints. It was made in Unreal Engine and can be used with an animation blueprint node.

Parallel

A school project made with a team of 20+ people, from 2nd to 4th year and a few master students. The project was mostly focussed on the managing and communicative side of game development. The game is a 4-player coop, top-down zombie shooter with a procedurally generated map and was made in Unreal Engine. I made the camera, score tracking, split-screen and loading/spawning of the buildings using the PCG locations.

3D Engine and Top-Down-Shooter Prototype

Bodemonderzoek

A university project for the education Environmental Science for Sustainable Energy and Technology (ESSET) where students walk through a recreated environment where they have to spot pollution in the soil. Group project consisting of six members, I made the camera with the detection and a feature to export the photo's with the walked route to a PDF.
